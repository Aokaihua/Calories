# Calories--卡路里小管家

本项目以练手为主要目的，涉及了自定义组件、父子组件传值、小程序页面生命周期等知识点，能够帮助新手快速熟悉如何从0到1创建一个微信原生版本小程序～

## 小程序功能

1. 搜索食物；
2. 计算摄入的卡路里，并以progress-bar的形式呈现在主页中；
3. 点击日历中的日期访问历史饮食记录
